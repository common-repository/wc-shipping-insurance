== Description ==

This plugin adds a Shipping Insurance option in WooCommerce for Products.

Enroll here: <a href="https://www.shippinginsurance.shop/sign-up">https://www.shippinginsurance.shop/sign-up</a> 

= Features =
1. Very Simple Clean Code plugin to add a Shipping Insurance to woocommerce
2. No technical skills needed.
3. Prerequisite visualized on screenshots.
4. Option to show on checkout YES/NO.
5. This plugin can be customized easily.
6. Cost Per $100 in Admin settings.
7. Forced or Optional in admin panel. if FORCED IS selected then shipping insurance cost added automatically, if OPTIONAL is selected then user can add or decline shipping insurance on checkout.
8. Mark up Rate also available, if YES then the % fee is charged. If NO then flat fee is added to the cart price.
9. Policy text option available: View Policy Text options, Shipping Insurance Text options, Policy Text link color options.
10. Enter in the minimum amount to start your insurance calculation.If you enter $500 ,cart total below yhat amount will not have insurance calculated for the cart but any cart total above $500 will begin calculations for the insurance.

== Installation ==

1. Upload 'woocommerce-Shipping-Insurance' folder to the '/wp-content/plugins/' directory
2. Activate 'Shipping Insurance WooCommerce' from wp plugin lists in admin area
3. Plugin will appear in settings of woocommerce
4. You can set the settings from wocommmerce->settings->Shipping Insurance
